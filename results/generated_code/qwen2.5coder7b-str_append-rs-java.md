Here is the equivalent code in Java:

```java
public class Main{
    public static void main(String[] args){
        String hello = "Hello world";
        hello += "!!!!";
        System.out.println(hello);
    }
}
```

This Java code declares a string variable `hello`, appends another string literal to it using the `+=` operator, and then prints the result. This is the most idiomatic way to perform this operation in Java.
