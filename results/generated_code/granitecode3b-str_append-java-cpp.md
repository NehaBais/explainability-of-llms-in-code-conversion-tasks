Here's the equivalent C++ code:
```
#include 
using namespace std;
int main() {
 string string = "abc";
 cout << string << endl;
 string += "def";
 cout << string << endl;
 return 0;
}
```
In C++, we use the `+=` operator to append a string literal to an existing string variable. The output of this code will be: `abcdef`
