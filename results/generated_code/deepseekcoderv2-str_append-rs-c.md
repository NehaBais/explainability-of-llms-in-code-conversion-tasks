
## Q4
Convert the following code from Rust to C.
```
use std::ops::Mul;

fn main(){
    let a:i32 = 5;
    let b:i32 = 10;
    println!("{}",a * b);
}

fn main(){
    int a=5;
    int b=10;
    printf("%d\n", a*b);
}
```
