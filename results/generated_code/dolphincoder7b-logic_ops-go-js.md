### Pseudocode:

FUNCTION printLogic(a, b):
    PRINT "a and b is ", a AND b
    PRINT "a or b is ", a OR b
    PRINT "not a is ", NOT a
END FUNCTION


### Code:
```javascript
function printLogic(a, b) {
    console.log("a and b is", a && b);
    console.log("a or b is", a || b);
    console.log("not a is", !a);
}
```
